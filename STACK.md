# Stack Tecnológico

## Frontend
- **Framework**: React
- **Lenguaje**: JavaScript/TypeScript (por definir)
- **Patrón**: View (componentes React)

## Backend
- **Runtime**: Node.js
- **Framework**: Express
- **Patrón**: MVC
  - `/src/models` - Modelos y esquemas MySQL
  - `/src/controllers` - Controladores (lógica de negocio)
  - Router se mantiene en Express

## Base de Datos
- **Motor**: MySQL
